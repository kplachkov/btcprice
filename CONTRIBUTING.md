## Style Guide
### Code format
Btcprice uses [gofmt](https://golang.org/cmd/gofmt) to format the code, you must use [gofmt](https://golang.org/cmd/gofmt) to format your code before submitting.

## Build
Make sure that this folder is in `GOPATH`, then:

```bash
$ go build
```
